Terminal

Task Shortcut Note
Open Terminal WINDOWS + T Check for windows / Linux OS logo on keyboard.
Open new tab CTRL + SHIFT + T Open new tab in terminal.
Close active tab CTRL + SHIFT + W Close the active tab in terminal.
Copy selected text CTRL + SHIFT + C Copy selected content to clipboard.
Paste from clipboard CTRL + SHIFT + V Paste the copied content
Open file open ./file_name & Opens the file with default associated app
Open file browser nautilus . & Opens the wrking directory in file browser
Create shortcuts ln -s //name //name Works for directories also
Overwrite shortcuts ln -sf //name //name Works for directories also
Search for files find ./path/to/file -type f -name 'file-name' Check for all the file with provided name
Search for directory find ./path/to/file -type d -name 'file-name' Check for all the directory with provided name
Search for text in all files find ./path/to/file -type f -name 'file-name' -exec grep --color -Hi 'text-to-search' {} \; Check for provided text in all matching files.
Search for text in a file grep --color -Hi 'text-to-search' *.txt Check for provided text in all matching files.
Open app in background gedit /path/to/file.txt & Open the file with gedit app in background process.
Open app in background nohup gedit /path/to/file.txt & Open the file in gedit app in background and logs app activity in nohup.out file.
Verify file integrity cksum /path/to/file Shows CRC checksum of file.
Check open port
Check running process
Kill running process
Update Specific app
Inspect system boot
